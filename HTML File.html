Interactive Visualization: NYC Squirrel Census Data

John Purcaro CSCE 567 Final Project: Squirrel Activity in Central Park, NYC

This interactive Tableau visualization provides insights into the NYC Squirrel Census data, highlighting patterns and trends in squirrel activities, locations, and behaviors. Use the tools provided to explore the data in detail and gain valuable insights.

The data used for this visualization was sourced from the NYC Squirrel Census dataset, available on the TidyTuesday GitHub repository .
